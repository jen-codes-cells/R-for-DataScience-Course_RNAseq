--- title: "Differential expression analysis of MOV10 over-expression in HEK293F cells" author: "Your Name" date: "`r Sys.Date()`" format:   html:     toc: true     toc-depth: 2     theme: cosmo     code-fold: false     output-file: index.html ---  In this study, we investigate the effects of **MOV10** overexpression in *HEK293F* cells using `DESeq2`.  ## Introduction  The data from this analysis is part of a larger study described in [Kenny PJ et al., Cell Rep 2014](https://pubmed.ncbi.nlm.nih.gov/25464849/).  ![](https://bioconductor.posit.co/images/logo/svg/Logo.svg)  The RNA-seq analysis was performed using the following steps:  1.  Quality control with [fastqc](https://www.bioinformatics.babraham.ac.uk/projects/fastqc/) 2.  Adapter trimming with [cutadapt](https://cutadapt.readthedocs.io/en/stable/) 3.  Pseudo-alignment with [salmon](https://combine-lab.github.io/salmon/) to the human transcriptome (GRCh38 Ensembl release 106).  The DESeq2 dataset contains counts for the following samples:  | Dataset    | Description           | Replicate   | |------------|-----------------------|-------------| | Control_1  | Control               | replicate 1 | | Control_2  | Control               | replicate 2 | | Control_3  | Control               | replicate 3 | | MOV10_OE_1 | MOV10 over-expression | replicate 1 | | MOV10_OE_2 | MOV10 over-expression | replicate 2 | | MOV10_OE_3 | MOV10 over-expression | replicate 3 |  ## Data analysis  ### Load libraries and dataset  ```{r load_libraries} #| warning: false #| message: false  # Load the libraries we will use in this analysis library(tidyverse) library(DESeq2) library(plotly) library(pheatmap) library(RColorBrewer) library(plotly) library(apeglm) library(DT) library(ggrepel)  # Load the DESeq2 dataset dds <- readRDS("data/MOV10_dds.rds") # Display the column data from our dataset colData(dds) ```  ### Run DESeq2  ```{r run_deseq2} ## run DESeq dds <- DESeq(dds) ```  ### Quality control  The dispersion estimates are used to model the variance in the data. The plot below shows the dispersion estimates for each gene in the dataset.  ```{r plot_dispersion} #| fig-cap: "Dispersion estimates for each gene in the dataset" #| fig-width: 6 #| fig-height: 4  #Plot dispersions plotDispEsts(dds, main="Dispersion plot") ```  The heatmap below shows the distances between samples in the dataset. The distance is calculated using the Euclidean distance between the samples based on the log-transformed counts.  ```{r heatmap_sample_distance} #| fig-cap: "Heatmap of sample distances" #| fig-width: 6 #| fig-height: 4  ## Log transformed data rld <- rlog(dds, blind=F)  sampleDists <- dist(t(assay(rld))) ## t() function transposes a matrix sampleDistMatrix <- as.matrix(sampleDists) rownames(sampleDistMatrix) <- as.list(colnames(dds)) colnames(sampleDistMatrix) <- as.list(colnames(dds)) cols <- colorRampPalette( rev(brewer.pal(9, "Blues")) )(255) ## Set a colour pallette in shades of blue pheatmap(sampleDistMatrix,          clustering_distance_rows = sampleDists,          clustering_distance_cols = sampleDists,          col = cols) ```  PCA analysis is a dimensionality reduction technique that allows us to visualize the variation in the dataset. The plot below shows the first two principal components of the dataset.  ```{r pca_plot} #| fig-cap: "PCA plot of the dataset" #| fig-width: 6 #| fig-height: 4  ## Principle component analysis - get the PCA data pca <- plotPCA(rld, intgroup = c("Group", "Replicate"), returnData = T)  ## Plot with ggplot pca <- ggplot(pca, aes(PC1, PC2, colour = Group, shape = Replicate, text = paste0(Group,"_",Replicate))) +    geom_point(size = 3) +   theme_bw() +    theme(legend.key = element_blank()) +    xlab(paste("PC1:",round(attr(pca,"percentVar")[1]*100),"%")) +    ylab(paste("PC2:",round(attr(pca,"percentVar")[2]*100),"%"))  pca ```  The plot below is an interactive version of the PCA plot above. You can hover over the points to see the sample names.  ```{r pca_plotly} #| warning: false #| message: false #| fig-cap: "Volcano plot MOV10 vs Control" #| fig-width: 6 #| fig-height: 6  ggplotly(pca, tooltip = "text") ```  The heatmap below shows the top 20 most variable genes in the dataset. The genes are clustered based on their expression patterns across the samples.  ```{r heatmap_top_genes} #| fig-cap: "Heatmap of the top 20 most variable genes" #| fig-width: 6 #| fig-height: 6  ## Get the top 20 genes after ordering the rld counts by variance topVarGenes <- head(order(-rowVars(assay(rld),useNames = T)), 20)  ## Create a matrix from these genes only mat <- assay(rld)[topVarGenes, ]  ## Create an annotation dataframe for the columns (samples) to show the Group and Replicate information anno <- as.data.frame(colData(dds)[,c("Group","Replicate")]) ## Plot the heatmap with pheatmap pheatmap(mat, cluster_rows = T, cluster_cols = T, show_rownames = T, scale = "row", annotation_col = anno) ```  ### Results MOV10 over-expression vs Control  The results below show the differential expression analysis between MOV10 over-expression and control samples.  We use the `lfcShrink` function to shrink the log2 fold changes for better visualisation and interpretation and set a significance threshold of 0.05 for the adjusted p-value.  ```{r results} #| warning: false #| message: false  ## Set significance threshold alpha = 0.05  ## Get the results for the comparison of interest resLFC <- lfcShrink(dds, coef = "Group_MOV10_OE_vs_Control", type = "apeglm")  ## Output a summary of results summary(resLFC, alpha = alpha) ```  The table below shows the results of the differential expression analysis. The results are sorted by the adjusted p-value (padj) in ascending order.  ```{r results_table} #| warning: false #| message: false  ## Read in the gene annotation file to get gene symbols for the Ensembl gene IDs gene_map <- read_tsv("data/salmon_tx2gene.tsv",col_names = c("TxID","GeneID","GeneSymbol")) |>   dplyr::select(GeneID, GeneSymbol) |>   unique()  ## Set a threshold for log2 fold change to call significant genes by effect size as well as padj lthresh = 1  ## Build the result table result_table <- resLFC |>     as.data.frame() |>      rownames_to_column("GeneID") |>      left_join(gene_map, by = "GeneID") |>      mutate(Significant = case_when(        padj <= alpha & abs(log2FoldChange) >= lthresh ~ "Both",       padj <= alpha & abs(log2FoldChange) < lthresh ~ "padj",       padj > alpha & abs(log2FoldChange) >= lthresh ~ "l2FC",       .default = "Not Significant")       ) |>     mutate(Significant = factor(Significant, levels = c("Not Significant", "padj", "l2FC", "Both"))) |>    arrange(padj) |>     as_tibble()  ## Output as datatable for interactive viewing DT::datatable(result_table,               options = list(pageLength = 10,                               scrollX = TRUE),                rownames = FALSE)                ```  ## Visualisation  #### Volcano Plot  The volcano plot below shows the log2 fold change (x-axis) versus the -log10 adjusted p-value (y-axis) for each gene in the dataset. The genes are coloured based on their significance.  ```{r volcano_plot} #| warning: false #| message: false #| fig-cap: "Volcano plot MOV10 vs Control" #| fig-width: 6 #| fig-height: 6  sig_cols <- c("Not Significant" = "grey60", "padj" = "#143960", "l2FC" = "#08A4BD", "Both" = "#C91F3A")  topTen <- result_table |> head(n = 10)  ggplot(result_table, aes(log2FoldChange, -log10(padj), colour = Significant)) +    geom_point(size = 1, alpha = 0.5) +    geom_text_repel(data = topTen, aes(label = GeneSymbol))+   theme_bw() +    scale_colour_manual(values = c(sig_cols)) +   ggtitle("MOV10_OE_vs_Control") +   guides(colour = guide_legend(override.aes = list(size=3, alpha = 1))) ```
